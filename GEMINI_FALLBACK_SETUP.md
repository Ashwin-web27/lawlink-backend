# Gemini AI Fallback Setup Guide

## 🎯 Overview

The chatbot now uses a **smart two-tier system**:

1. **Primary**: Searches your legal document database first
2. **Fallback**: Uses Gemini AI as a lawyer assistant when dataset doesn't have good answers

## 🔧 Setup Instructions

### Step 1: Get Gemini API Key

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your API key

### Step 2: Configure Environment Variable

Create a `.env` file in the `backend/` directory:

```bash
cd backend
touch .env  # Linux/Mac
# or create .env file manually on Windows
```

Add your API key:

```env
GEMINI_API_KEY=your_actual_api_key_here
```

**Important**: 
- Never commit the `.env` file to git
- Keep your API key secure
- The `.env` file should be in `.gitignore`

### Step 3: Install Dependencies

```bash
cd backend
pip install -r requirements.txt
```

This will install `google-generativeai` package.

### Step 4: Restart FastAPI Server

```bash
python src/api/main.py
```

You should see in the logs:
```
✅ Gemini service initialized successfully with model: gemini-1.5-flash
```

## 🔄 How It Works

### Flow Diagram:

```
User Question
    ↓
Search Legal Document Database
    ↓
Calculate Confidence Score
    ↓
┌─────────────────────────────┐
│ Confidence ≥ 30%?          │
│ AND Results Found?         │
└─────────────────────────────┘
    ↓ YES              ↓ NO
Use Dataset      Use Gemini AI
Answer           (Lawyer Assistant)
    ↓                  ↓
Return Answer    Return Answer
with Sources     (AI Generated)
```

### Confidence Threshold

- **Default**: 30% (0.3)
- **Location**: `backend/src/api/main.py` - `CONFIDENCE_THRESHOLD = 0.3`
- **Adjustment**: You can change this value to be more/less strict

### Example Scenarios:

**Scenario 1: Question in Dataset (High Confidence)**
- Question: "What is Article 21?"
- Dataset finds: Article 21 from Constitution
- Confidence: 0.95 (95%)
- **Result**: Uses dataset answer with sources

**Scenario 2: Question Not in Dataset (Low Confidence)**
- Question: "What are the latest changes in property law?"
- Dataset finds: Some related articles but low match
- Confidence: 0.15 (15%)
- **Result**: Falls back to Gemini AI for comprehensive answer

**Scenario 3: No Results in Dataset**
- Question: "How do I file a patent in India?"
- Dataset finds: No results
- Confidence: 0.0 (0%)
- **Result**: Uses Gemini AI to provide helpful answer

## ⚙️ Configuration Options

### Change Confidence Threshold

Edit `backend/src/api/main.py`:

```python
# Line ~245
CONFIDENCE_THRESHOLD = 0.3  # Change to 0.2 for more lenient, 0.5 for stricter
```

### Change Gemini Model

Edit `backend/src/services/gemini_service.py`:

```python
# Line ~45
self.model = genai.GenerativeModel("gemini-1.5-flash")
# Change to "gemini-2.0-flash-exp" if you have access
```

### Customize Gemini Prompt

Edit `backend/src/services/gemini_service.py`:

```python
# Line ~30-40
system_prompt = """Your custom prompt here..."""
```

## 🐛 Troubleshooting

### Issue: "GEMINI_API_KEY not found in environment variables"

**Solution:**
1. Check that `.env` file exists in `backend/` directory
2. Verify the file contains: `GEMINI_API_KEY=your_key`
3. Restart the FastAPI server
4. Check that `python-dotenv` is installed: `pip install python-dotenv`

### Issue: "google-generativeai not installed"

**Solution:**
```bash
pip install google-generativeai
```

### Issue: "Error initializing Gemini"

**Possible Causes:**
1. Invalid API key - Check your API key is correct
2. API quota exceeded - Check your Google Cloud quota
3. Network issues - Check internet connection

**Solution:**
- Verify API key at: https://makersuite.google.com/app/apikey
- Check API usage in Google Cloud Console
- The chatbot will still work without Gemini (using dataset only)

### Issue: Gemini always used even for dataset questions

**Solution:**
- Check confidence threshold setting
- Lower threshold if dataset answers are being ignored
- Check dataset search is working: `curl https://llm.skyzin.com/api/search -d '{"query":"Article 21"}'`
